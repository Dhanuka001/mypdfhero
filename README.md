# MyPDFHero

Monorepo for MyPDFHero, a login-free PDF utilities site. Contains the Next.js frontend, Express backend, and infra tooling.

## Structure
- `frontend/` – Next.js + Tailwind app router UI
- `backend/` – Express API for PDF operations
- `infra/` – Dockerfiles, compose, proxy config, CI scripts

See `ARCHITECTURE.md` for high-level design.
